# recursive-reverse

> Recursively reverse a string: `hello world` → `dlrow olleh`

## Install

```
$ npm install recursive-reverse
```

## Usage

```js
const recursiveReverse = require('recursive-reverse');

recursiveReverse('hello world');
//=> 'dlrow olleh'
```
